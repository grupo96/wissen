Linha do tempo do desenvolvimento de modelos de RNA

A jornada para construir um modelo de classificação.

Sintaxe dos Passos

O objetivo deste trabalho é desenvolver um modelo de rede neural capaz de classificar os perfis de aprendizagem em quatro categorias: Visual, Auditivo, Cinestésico e Digital. O resultado dessa classificação será utilizado para ajustar os conteúdos e abordagens de ensino de acordo com o perfil do aluno identificado.

Além disso, estão sendo implementados modelos GPT com a função de auxiliar professores e monitores na geração de conceitos e exercícios personalizados para os alunos cadastrados em suas respectivas turmas. O objetivo é proporcionar uma experiência de aprendizagem mais dinâmica e alinhada ao estilo de cada estudante.

Abaixo segue uma linha do tempo, onde é mostrado todos os passos

Outubro de 2024

  • Criamos um formulário para coletar perfis de aprendizagem.

  • Implementamos critérios para classificar os perfis em Visual, Auditivo, Cinestésico e Digital.

  • Desenvolvemos scripts para automatizar o processo em Python.

Novembro de 2024

  • Usamos k-NN como primeiro modelo para realizar as classificações dos prefis.

  • Para a classificação dos perfis de aprendizagem, também foi utilizado o modelo de rede neural MLP (Multi-Layer Perceptron), que demonstrou ser eficaz ao combinar múltiplas camadas de processamento para analisar e classificar os dados com alta precisão.

  • Testamos diferentes técnicas de normalização, optando pelo StandardScaler.

  • Ajustamos os dados para otimizar o desempenho dos modelos.

Meados de Novembro de 2024

  • Com 47 amostras, utilizamos GANs para gerar dados sintéticos.

  • GANs (Redes Generativas Adversariais) são modelos de aprendizado de máquina compostos por duas redes neurais que competem entre si: uma geradora, que cria dados falsos, e uma discriminadora, que tenta identificar se os dados são reais ou falsos. No nosso caso, utilizamos GANs para gerar dados sintéticos, ajudando a aumentar o tamanho do conjunto de dados disponível.

  • Obtivemos uma acurácia de 99,39% em uma MLP treinada com os dados gerados.

Final de Novembro de 2024

  • Exploramos modelos generativos como GPT-J e Falcon-7B.

  • Superamos problemas de hardware e ajustes no tokenizador.

  • Testamos a viabilidade para geração de conteúdo educacional.

Status Atual

  • Integração de GPTs para personalizar conteúdos teóricos e exercícios.

  • Foco em aprendizagem adaptativa na plataforma Wissen.
